Instructions

Install

Python 2.7.x
setuptool
Flask-0.10.1
Jinja2-2.7
Werkzeug-0.9.3

Copy fibowebservices to C:\
at commandline run python fibowebservices.py
In browser run http://127.0.0.1:5000/fibonacci/number
or
Install Curl
at commandline run
ex. C:\Curl curl http://127.0.0.1:5000/fibonacci/5

xsd(xml file) will be written to the C:\fibowebservices\ directory
